# SIAP-
SIAP空中态势仿真软件
SIAP空中态势仿真软件 SIAP-Simulator 是一个面向空中指挥控制系统研发的轻量级全栈仿真测试平台，基于 Python + WebSockets + HTML5 Canvas 构建。该系统通过后端动态模拟多源传感器（GCI/AWACS）对空中目标的探测过程，并利用 WebSocket 将模拟报文实时推送至前端；前端则基于原生 Canvas 开发了高性能态势渲染引擎，实现了地理坐标映射、图层交互以及多源信息融合（SIAP）核心算法的可视化，支持 IFF 投票、航迹关联与置信度评估的全流程展示，并内置了交互调试工具链，将前端开发与联调的环境搭建时间从小时级大幅缩短至分钟级。
